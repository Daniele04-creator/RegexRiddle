# RegexRiddle Specification

## GOAL 00 scope

This milestone creates the initial repository skeleton only. It proves that the project can be installed, checked, built, tested, and started locally before product features are added.

## Official project requirements

- Modern and secure full-stack web application.
- Backend API implemented with a web framework.
- Responsive frontend SPA implemented with a web framework.
- At least 10 automatic E2E tests by final delivery.
- Frontend and backend kept in distinct directories.
- No CMS.
- Detailed README.
- No `node_modules` in the final ZIP.

## Product idea

RegexRiddle will become a regex puzzle game. Solvers will submit candidate strings or answers without seeing secret challenge regexes or secret controls.

## Explicitly out of scope for GOAL 00

- Authentication.
- Prisma schema or database migrations.
- Challenge creation.
- Attempt engine.
- Leaderboard.
- Regex evaluation.
- Final UI system.
